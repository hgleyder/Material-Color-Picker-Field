#
Without Value

```jsx
<ColorPickerField 
    label="Example color"
    placeholder="Pick a color"
/>
```
#
With Value

```jsx
<ColorPickerField 
    label="Example color"
    value="#f50057"
/>
```
#
With Value (full width)

```jsx
<div style={{width: '100%'}}>
    <ColorPickerField 
        label="Example color"
        value="#f50057"
        fullWidth
    />
</div>
```
